Post it board using React and Node.js

User can add post-its to a stored collection.

Start project from terminal

Frontend:
git init https://github.com/mcrayman/Post-it-Board.git
npm install
npm run dev

Backend: 
git init https://github.com/mcrayman/Backend-support-for-Post-it-Board.git
npm install
npm start
